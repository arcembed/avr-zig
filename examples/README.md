# Examples

Each example is a standalone Zig project that depends on the root `avr_zig` package through a local path dependency.

Available examples:

- `analog-input`
- `blink`
- `button`
- `custom-hooks`
- `dht11`
- `hc-sr04`
- `i2c-scan`
- `lcd-1602-i2c`
- `pwm-fade`
- `pwm-rgb`
- `servo`
- `ssd1306-demo`
- `uart`
